### Objetivo da Atividade
Criar um jogo interativo de adivinhação de número secreto utilizando **HTML, CSS e JavaScript puro**, com as seguintes exigências avaliativas:

- Geração de número aleatório entre 1 e 100
- Limite de 10 tentativas
- Validação completa do input
- Feedback claro ao usuário (maior/menor/acertou)
- Contador de tentativas restantes
- Uso da biblioteca **SweetAlert2** para mensagens de boas-vindas e fim de jogo
- Botão "Jogar Novamente" funcional
- Código limpo, organizado e responsivo

**Nota: Todos os requisitos foram implementados com sucesso!**
